# 纳斯达克场外定投量化交易体系 (CNN 恐惧与贪婪指数增强版)

## 一、 体系定位
本体系将 **CNN 恐惧与贪婪指数** 作为“全市场风险偏好确认器”，与 VXN/VIX（波动率恐慌）、NDXE/SOX（纳指内部结构）及 SMA200（趋势门控）共同构成多维择时模型。

*   **VXN/VIX**：看“波动率恐慌”。
*   **CNN 贪婪指数**：看“全市场风险偏好”。
*   **NDXE/SOX**：看“纳指内部结构”。
*   **SMA200**：看“趋势环境”。

---

## 二、 核心架构：四层过滤
加入 CNN 指数后，体系分为以下四个层级：

| 层级 | 指标 | 作用 |
| :--- | :--- | :--- |
| **第一层：趋势门控** | NDX/QQQ vs SMA200 ±5% | 决定是否具备激进补仓的基础环境 |
| **第二层：情绪确认** | VXN、VIX、CNN Fear & Greed | 判断市场处于恐慌、中性还是贪婪状态 |
| **第三层：结构确认** | NDXE/NDX、SOX/NDX | 判断纳指内部是共振修复还是背离走弱 |
| **第四层：执行计划** | 每天200元，3/5/10/20天窗口 | 适配场外定投申购限额与确认滞后 |

---

## 三、 CNN 指数实战分区与含义
CNN 指数用于区分“局部恐慌”与“全市场共振”，作为 VXN/VIX 的重要确认。

| CNN 区间 | 情绪状态 | 对定投的指导意义 |
| :--- | :--- | :--- |
| **0–25** | 极度恐惧 | 进入补仓观察区（猎物区） |
| **26–44** | 恐惧 | 提高关注，观察是否企稳回升 |
| **45–55** | 中性 | 正常定投节奏 |
| **56–74** | 贪婪 | 降低追高冲动，减少定投频率 |
| **75–100** | 极度贪婪 | 进入暂停/降频观察区 |

**核心规则：**
1. **CNN 低位不急买**：等待从极度恐惧区（<25）开始回升。
2. **CNN 高位不急卖**：先降频/暂停新增，观察结构是否背离。

---

## 四、 评分系统 (Buy & Sell Score)

### 1. Buy Score (决定补仓窗口长度)
总分 100 分。用于判断是否开启 200元/天 的加速补仓模式。

| 模块 | 条件 | 分数 |
| :--- | :--- | :--- |
| **波动率恐慌** | VXN 或 VIX 高于 80% 历史分位 | +15 |
| **波动率转折** | VXN/VIX 从高位开始回落 | +15 |
| **CNN 恐惧** | CNN 指数 < 25 (或 < 35) | +15 |
| **CNN 修复** | CNN 从低位回升，或 5 日均值上行 | +15 |
| **广度修复** | NDXE/NDX 止跌或上穿 20 日均线 | +15 |
| **主线修复** | SOX/NDX 止跌或上穿 20 日均线 | +15 |
| **趋势保护** | NDX > 0.95 × SMA200 或重回缓冲带 | +10 |

**执行标准：**
*   **< 40分**：不加速，维持正常/低速定投。
*   **40–60分 (轻补)**：开启 **3–5个交易日** 200元/天窗口。
*   **60–75分 (标准补)**：开启 **10个交易日** 200元/天窗口。
*   **> 75分 (深补)**：开启 **20个交易日** 200元/天窗口。
*   *特殊：若 NDX < 0.95M 且 SMA200 向下，不开 20 天窗口，仅做试探性补仓。*

### 2. Sell Score (决定降频、暂停或处理战术仓)
总分 100 分。用于判断市场过热程度。

| 模块 | 条件 | 分数 |
| :--- | :--- | :--- |
| **价格偏热** | NDX > 1.20 × SMA200 | +10 |
| **价格过热** | NDX > 1.25 × SMA200 | +15 |
| **CNN 贪婪** | CNN 指数 > 75 (极度贪婪) | +20 |
| **CNN 转弱** | CNN 从高位回落，或 5 日均值下行 | +10 |
| **波动率低位** | VXN/VIX 处于历史极低分位 | +15 |
| **广度背离** | NDX 创新高但 NDXE/NDX 走弱 | +15 |
| **主线背离** | NDX 创新高但 SOX/NDX 走弱 | +15 |

**执行标准：**
*   **< 40分**：正常定投。
*   **40–60分 (降频)**：降低定投频率（如每周 1-2 次），持续 10 天。
*   **60–75分 (暂停)**：暂停所有新增定投 10–20 个交易日。
*   **> 75分 (预警)**：暂停新增 20 天，考虑每周赎回战术仓 5%–10%。

---

## 五、 执行档位定义

| 档位 | 条件 | 核心操作 |
| :--- | :--- | :--- |
| **防守档** | Sell Score > 60 | 暂停新增 10–20 个交易日 |
| **低速档** | CNN 贪婪但结构未坏 | 每周 1–2 次，每次 200 元 |
| **正常档** | CNN 中性，指标平稳 | 隔日 200 元，或 100 元/天 |
| **轻补档** | Buy Score 40–60 | 200 元/天 × 3–5 天 |
| **标准补仓档** | Buy Score 60–75 | 200 元/天 × 10 天 |
| **深度补仓档** | Buy Score > 75 | 200 元/天 × 20 天 |

## 六、 关键实战场景

### 1. 最强补仓信号 (共振回升)
*   **条件**：CNN < 25 后回升 + VXN 高位回落 + NDXE/SOX 止跌 + 趋势未恶化。
*   **策略**：果断开启 **10–20天** 200元/天补仓窗口。

### 2. 只观察不急补 (自由落体)
*   **条件**：CNN < 25 且继续下跌 + VXN 持续飙升 + 结构走弱 + 跌破 0.95M。
*   **策略**：不做深度补仓，等待恐慌情绪出清，维持极低频定投。

### 3. 高位风险确认 (背离过热)
*   **条件**：CNN > 75 + NDX 严重偏离 SMA200 + VXN 极低 + NDXE/SOX 背离。
*   **策略**：立即**停止追高**，暂停新增，等待情绪回归。

---

## 七、 执行规则
1. **每周判断一次**：周末复盘各指标，定下周执行计划。
2. **CNN 与 VXN 共振原则**：两者同时恐慌时，补仓价值最高；两者同时贪婪时，风险最高。
3. **先停后赎**：高位时先执行“暂停新增”，确认趋势反转后再处理战术仓。核心仓长线持有。
4. **现金流管理**：确保账户现金储备足以支撑 20 天连续补仓（约 4,000 元）。
